# 更新日志

## v1.1.0 - 2024-11-10

### 新增功能
- ✨ 添加提示词管理系统，支持从文件加载提示词
- ✨ 支持多个 AI 提供商：Kimi AI 和 Ollama
- ✨ 使用官方 SDK 替代手动 HTTP 请求
  - Kimi AI: 使用 OpenAI SDK
  - Ollama: 使用 Ollama Python SDK
- ✨ 提示词装饰器，支持自动注入系统提示词
- ✨ 优化标签和描述提取逻辑，现在从完整的阅读指南中提取

### 改进
- 🔧 重构 AI 服务层，使用策略模式支持多提供商
- 🔧 改进配置管理，支持通过环境变量切换 AI 提供商
- 📝 完善提示词系统，支持缓存和变量替换
- 📝 更新文档和示例配置

### 技术栈更新
- 添加 `openai` SDK（用于 Kimi AI）
- 添加 `ollama` SDK（用于 Ollama）
- 移除 `httpx`（不再需要手动 HTTP 请求）

## v1.0.0 - 2024-11-10

### 初始版本
- 🎉 项目初始化
- ✨ 实现文献上传、解析和管理
- ✨ 集成 AI 生成阅读指南
- ✨ 支持多种文件格式：PDF、Word、Markdown
- ✨ SSE 流式响应
- ✨ 完整的 REST API
